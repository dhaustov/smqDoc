Именование полей классов:
Если храним id то префикс "id"
Если храним Объект префикс "obj"
Если храним массив префикс "lst"
Методы все с большой буквы.
Обязательно указывать тип поля/возвращаемого значения для элементов класса

Возвращаемое значени либо false - Ошибка/Пустой результат

Репозиторий:
Имена таблиц заданы в начале класса константами вида: TBL_ИМЯТАБЛИЦЫ
Методы save меняют значения поле id передаваемого объекта на записанный в базе


Объекты:
Если объект представляет сущность таблдицы связи - то имя: перваяСущность_ВтораяСущность
